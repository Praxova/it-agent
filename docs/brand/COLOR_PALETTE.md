## Brand Identity

### Visual Identity
The IT Agent brand embodies professional enterprise IT automation through a geometric, minimalist robot/AI assistant icon. The design features clean lines and integrated circuit board patterns that communicate both technological sophistication and approachability. The symmetrical robot head suggests reliability and precision — core values for enterprise IT operations.

### Color Psychology
- **Deep navy blue (#0A2463)** — Trust, stability, professionalism. Essential for enterprise IT handling sensitive operations (AD changes, password resets).
- **Bright cyan/teal (#00B4D8)** — Energy and modernity. Represents AI innovation and intelligent automation.
- **Silver/white (#E8F1F5)** — Clarity and readability across interfaces.

### Usage Guidelines
- **Logo placement**: Full-color icon on light backgrounds; reversed for dark mode
- **Minimum size**: 32x32px to maintain circuit board detail visibility
- **Clear space**: 10% of icon height padding on all sides
- **Best backgrounds**: White, light gray (#F5F5F5), or navy blue (#0A2463)
- **Accessibility**: All color combinations meet WCAG AA contrast requirements
